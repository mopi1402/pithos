@keywords: head, first, element, array

Get the first element of an array.

```typescript
const items = [1, 2, 3, 4];
items[0];
// => 1
```

### **Get** element at position

@keywords: nth, index, position, element

Access element at a specific index.

```typescript
const items = ["a", "b", "c", "d"];
items[2];
// => "c"
```

### **Get** element from end

@keywords: negative, index, last, end

Access elements from the end using negative index.

```typescript
const items = [1, 2, 3, 4, 5];
items.at(-2);
// => 4
```
