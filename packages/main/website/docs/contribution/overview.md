## üéØ Philosophy

> _"The need guides me. Technology follows."_

Everything in Pithos stems from this principle. We don't choose a technology because it's trendy ‚Äî we choose it because it solves a real problem.

**Priority hierarchy:**
1. **User Experience (UX)** ‚Äî Performance, bundle size, time to interactive
2. **Developer Experience (DX)** ‚Äî Intuitive API, clear docs, explicit errors
3. **Pragmatism** ‚Äî Smart compromises where it matters

See [Core Philosophy](../basics/core-philosophy.md) for the foundational vision.
